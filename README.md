# go-zerotier

Go Wrapper for [Zerotier SDK](https://github.com/zerotier/libzt/)

### Build Examples
```
go get -v github.com/samhug/go-libzt
cd $GOPATH/src/github.com/samhug/go-libzt
cd examples
CGO_LDFLAGS_ALLOW='-std=c\+\+11' go build
```
